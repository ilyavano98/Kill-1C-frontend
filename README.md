Установка и запуск React.js
1. Установить Node.js - официальная ссылка для виндодрочеров https://nodejs.org/en/download
2. После установки в git bash консоли (удобная штука, совтую пользоваться) или в любой другой
удобной консоли, жмякаем команду ***npm -v*** для проверки что Node.js установлена.
3. Переходим в дирректорию /frontend и выполняем в ней команду
***npm install***, так как все что нужно для скачивания прописал в package.json
4. Для запуска приложухи реакта локально пишем в консоли ***npm start*** находясь в /frontend
